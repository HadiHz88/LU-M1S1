# React Mock Exam Template

This is a mock exam template based on React Tutorial Chapters 01-09.

## Topics Covered

- Chapter 1: Introduction to React
- Chapter 2: Getting Started
- Chapter 3: JSX
- Chapter 4: Components
- Chapter 5: Props
- Chapter 6: State
- Chapter 7: Hooks (useState)
- Chapter 8: Events
- Chapter 9: Forms

## Setup Instructions

1. **Install Dependencies**

   ```bash
   npm install
   ```

2. **Start Development Server**

   ```bash
   npm run dev
   ```

3. **Open Browser**
   - The app will open at `http://localhost:5173`

## How to Use

1. **Add your information**: Add your Student ID and Full Name as comments at the top of `src/App.jsx`
2. Read the questions in `exam.md`
3. Implement your solutions by modifying the template in `src/App.jsx` or create separate component files in `src/`
4. Test your solutions in the browser using `npm run dev`
5. Make sure your code follows React best practices

## Submission

1. Compress the entire `React_Mock_Exam` folder (including `node_modules`)
2. Name the file: `[YourStudentID]_React_Exam.zip`
3. Upload to the URL provided during the exam

## Dependencies

- **React 18.2.0** - React library
- **React DOM 18.2.0** - React DOM rendering
- **Vite 5.0.8** - Build tool and dev server
- **@vitejs/plugin-react 4.2.1** - Vite plugin for React

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build

## Notes

- This template provides a basic structure to get you started
- You can create additional component files in the `src` folder as needed
- Focus on clean, readable code and React best practices
- Make sure to test your solutions before submitting
